What is this project about?

This project is about creating a simple HTTP server in Python that serves 3 static web pages.

This server should:

  • Handle basic HTTP requests
  • Send 200/404 HTTP responses
  • Serve static pages with animations and appropriate MIME types
  • Logging requests

Where does the server respond?

The server responds on http://localhost:8080 by default. This is written in the requirements of the project. But you can change the host and port by modifying the code, if you wanted...
